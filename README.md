# Посик пользователей

## Установка проекта
Склонируйте репозиторий:

```bash
git clone https://github.com/yato90/search_users.git
```

## Настройка Бекенда:

```bash
cd back
npm install
npm run build
npm run start
```

## Настройка Фронта:

```bash
cd front
npm install
npm run dev
```

После чего можно смотреть по: http://localhost:5173/
